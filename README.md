🚀 نظرة عامة على المشروع (Arabic)
​هذا المشروع هو قبو ذكي تم تطويره باستخدام لغة Rust، ويهدف إلى توفير بيئة آمنة لتخزين النصوص وتشفيرها. يتميز البرنامج بواجهة تفاعلية ملونة وقدرة على تحليل حالة المستخدم وتذكر عدد الأسرار المحفوظة.
​المميزات الرئيسية:
​نظام وصول آمن: يتطلب مفتاح دخول خاص لحماية البيانات.
​عداد الأسرار: يقوم بقراءة عدد السجلات المخزنة وإظهارها للمستخدم.
​تشفير البيانات: يستخدم خوارزمية تشفير خاصة لضمان عدم قراءة النصوص من قبل الغرباء.
​تحليل المشاعر: نظام تفاعلي يستجيب بناءً على الكلمات المفتاحية للمستخدم.
​🚀 Technical Overview (English)
​Ranger AI Vault is a secure data storage system built with Rust. It focuses on data privacy through custom encryption and provides an intelligent interactive terminal experience.
​Key Features:
​Security Gateway: Access is strictly controlled via a secure authentication key (Ranger2024).
​Smart Log Analytics: Automatically parses the storage file to count and display the number of secured entries.
​Custom Cipher Encryption: Implements a character-shifting encryption algorithm to maintain data confidentiality.
​Sentiment Interaction: A logic engine that provides dynamic responses based on user input and mood detection.
​Optimized CLI: Built using ANSI escape codes for a professional, color-coded terminal interface.
